PYTHON DATA CLEANING PANDAS
Cleaned and preprocessed the Global Superstore dataset using Python and Pandas by handling missing values, removing duplicates, converting data types, and exporting a cleaned dataset.
 🧹 Data Cleaning with Pandas

A Python project demonstrating data cleaning and preprocessing techniques using the Pandas library on the Global Superstore dataset.



 📌 Project Overview

This project focuses on preparing raw data for analysis by applying common data cleaning techniques.

Tasks completed include:

- Loading the dataset
- Identifying missing values
- Handling missing values
- Removing duplicate records
- Converting data types
- Exporting the cleaned dataset

---

 📂 Dataset

Dataset: Global Superstore

Files included:

- Global_Superstore.csv (Original)
- Global_Superstore_Cleaned.csv (Cleaned)

1. Imported the dataset using Pandas.
2. Checked dataset information.
3. Identified missing values.
4. Removed or handled missing values.
5. Checked for duplicate rows.
6. Removed duplicate records.
7. Converted string columns to appropriate data types.
8. Exported the cleaned dataset as a CSV file.
